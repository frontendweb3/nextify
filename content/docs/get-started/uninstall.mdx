Follow these steps to completely to remove nextify and its components from your server.

<Steps>

<Step>

### If you use npm, follow the given steps to uninstall nextify.

```bash title="npm"
npm uninstall fumadocs-core fumadocs-mdx fumadocs-ui
```
</Step>
 
<Step> 
### If you use pnpm, follow the given steps to uninstall nextify.
```bash title="pnpm"
pnpm rm fumadocs-core fumadocs-mdx fumadocs-ui
```
</Step>

<Step>
### If you use yarn, follow the given steps to uninstall nextify.
```bash title="yarn" 
yarn remove fumadocs-core fumadocs-mdx fumadocs-ui
```
</Step>

<Step> 
### If you use bun, follow the given steps to uninstall nextify.
```bash title="bun"
bun remove  fumadocs-core fumadocs-mdx fumadocs-ui
```
</Step>
</Steps>
